CellTool (ct)

一个模块化的命令行工具集，用于辅助日常工作。工具集采用命令组（command group）管理，每个工具都是独立的微应用，可以单独使用或集成到其他应用中。

特点

- 模块化设计，工具之间完全解耦

- 支持多层命令结构（ct -> command group -> command）

- 每个工具都可以独立运行

- 支持GUI集成

- 统一的文档管理

- 使用Python Click框架，提供友好的命令行界面

CellTool框架的结构 ```

celltool/

├── ct.py                 # 主入口

├── commands.yaml        # 命令配置（自动生成）

├── pyproject.toml      # 项目配置

```
